System Scripts

Windows 10 25H2 Hillcom Edition

Background scripts and registry tweaks applied automatically during setup to optimise and clean up your Windows installation.

  • Registry Tweaks: Applied once on setup
  • Bloat Removal: Runs on every logon
  • Pause Updates: Runs on every boot
  • All scripts: Can be disabled or removed

Registry Tweaks

A set of registry entries applied once during setup via ApplyToDefault.reg. These tweaks adjust system behaviour and visual settings for a better out-of-the-box experience.

Applied Tweaks
  • AutoColorization — Accent colour automatically picked from your wallpaper
  • MenuShowDelay — Reduced to 100ms for snappier context menus
  • DisableSearchBoxSuggestions — Removes Bing web suggestions from File Explorer search
  • EnableTransparency — Enables Fluent/Acrylic transparency effects
  • LaunchTo (This PC) — File Explorer opens to This PC instead of Quick Access
  • TaskbarSmallIcons — Taskbar uses small icons for a more compact layout
  • StartupDelayInMSec — Explorer startup delay reduced to 1ms for faster login
  • NumLock on login — Num Lock is automatically enabled at the login screen
How to Revert — Command Prompt

Open Command Prompt as Administrator and run the commands for any tweaks you want to undo:

rem AutoColorization — disable
reg add "HKCU\Control Panel\Desktop" /v AutoColorization /t REG_DWORD /d 0 /f

rem MenuShowDelay — restore to Windows default (400ms)
reg add "HKCU\Control Panel\Desktop" /v MenuShowDelay /t REG_SZ /d 400 /f

rem DisableSearchBoxSuggestions — remove (restores Bing suggestions)
reg delete "HKLM\SOFTWARE\Policies\Microsoft\Windows\Explorer" /v DisableSearchBoxSuggestions /f

rem EnableTransparency — disable transparency
reg add "HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize" /v EnableTransparency /t REG_DWORD /d 0 /f

rem LaunchTo — restore to Quick Access (default)
reg add "HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced" /v LaunchTo /t REG_DWORD /d 2 /f

rem TaskbarSmallIcons — restore normal icon size
reg add "HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced" /v TaskbarSmallIcons /t REG_DWORD /d 0 /f

rem StartupDelayInMSec — remove (restores default delay)
reg delete "HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Serialize" /v StartupDelayInMSec /f

rem NumLock — disable NumLock at login
reg add "HKCU\Control Panel\Keyboard" /v InitialKeyboardIndicators /t REG_SZ /d 0 /f
reg add "HKU\.DEFAULT\Control Panel\Keyboard" /v InitialKeyboardIndicators /t REG_SZ /d 0 /f
How to Revert — PowerShell

Open PowerShell as Administrator and run:

# AutoColorization — disable
Set-ItemProperty -Path "HKCU:\Control Panel\Desktop" -Name "AutoColorization" -Value 0

# MenuShowDelay — restore default (400ms)
Set-ItemProperty -Path "HKCU:\Control Panel\Desktop" -Name "MenuShowDelay" -Value "400"

# DisableSearchBoxSuggestions — remove
Remove-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows\Explorer" -Name "DisableSearchBoxSuggestions" -Force

# EnableTransparency — disable
Set-ItemProperty -Path "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize" -Name "EnableTransparency" -Value 0

# LaunchTo — restore Quick Access
Set-ItemProperty -Path "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced" -Name "LaunchTo" -Value 2

# TaskbarSmallIcons — restore normal size
Set-ItemProperty -Path "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced" -Name "TaskbarSmallIcons" -Value 0

# StartupDelayInMSec — remove
Remove-ItemProperty -Path "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Serialize" -Name "StartupDelayInMSec" -Force

# NumLock — disable at login
Set-ItemProperty -Path "HKCU:\Control Panel\Keyboard" -Name "InitialKeyboardIndicators" -Value "0"
Set-ItemProperty -Path "Registry::HKU\.DEFAULT\Control Panel\Keyboard" -Name "InitialKeyboardIndicators" -Value "0"
⚠ Important: Hillcom is not responsible for any damage, instability, or data loss caused by editing the Windows registry. Registry changes take effect after restarting Explorer or rebooting. Proceed only if you are confident in what you are doing. When in doubt, create a registry backup first via regedit → File → Export.

Bloat Removal

A PowerShell script (BloatRemoval.ps1) registered as a scheduled task that runs automatically on logon. It removes pre-installed Microsoft apps that most users do not need, keeping the system clean and lightweight.

Apps Removed
  • Bing News & Bing Weather
  • Get Help
  • Windows Dev Home
  • Microsoft Family
  • Feedback Hub
  • Microsoft People
  • Xbox Identity Provider & TCUI
  • Get Started
  • Microsoft Copilot
  • Microsoft Office Hub
  • Windows Recall (optional feature)
How to Disable the Scheduled Task
  1. Press Win + R, type taskschd.msc and press Enter
  2. In the left panel, expand Task Scheduler Library → Winhance
  3. Right-click BloatRemoval and choose Disable or Delete
How to Remove the Script
  1. Delete the file at C:\ProgramData\Winhance\Scripts\BloatRemoval.ps1
  2. Optionally delete the log folder at C:\ProgramData\Winhance\Logs\

Pause Windows Update

A PowerShell script (PauseWindowsUpdate.ps1) registered as a scheduled task that runs on every boot. It automatically pauses both feature and quality Windows Updates for 7 days, preventing unexpected update interruptions.

Key Details
  • Runs daily on every system boot
  • Pauses feature updates and quality updates for 7 days from current date
  • Writes pause timestamps to HKLM\SOFTWARE\Microsoft\WindowsUpdate\UX\Settings
  • Updates can still be installed manually via Windows Update Settings at any time
How to Disable the Scheduled Task
  1. Press Win + R, type taskschd.msc and press Enter
  2. Search for the task named PauseWindowsUpdate
  3. Right-click it and choose Disable or Delete
How to Resume Updates Immediately
  1. Open Settings → Update & Security → Windows Update
  2. Click Resume updates
