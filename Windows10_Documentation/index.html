Browsers & Internet

Apps for browsing the web and accessing remote machines.

Google Chrome

Google Chrome is a fast, secure, and widely-used web browser developed by Google. It offers a clean interface, powerful developer tools, and seamless integration with Google services like Gmail, Drive, and YouTube.

Key Features
  • Built-in password manager and autofill
  • Sync bookmarks, history, and settings across devices via Google account
  • Vast library of extensions from the Chrome Web Store
  • Built-in PDF viewer and reader mode
  • Automatic updates and strong security sandboxing
  • Fast JavaScript engine (V8) for snappy web app performance

AnyDesk

AnyDesk is a lightweight and fast remote desktop application that lets you access and control another computer from anywhere. It is commonly used for IT support, remote work, and file transfers between machines.

Key Features
  • Low-latency remote access with high-quality display
  • File transfer between local and remote machines
  • Unattended access support for servers and home PCs
  • Cross-platform — connects to Windows, macOS, Linux, Android, and iOS
  • Session recording and audit log
  • No installation required on the remote side (portable mode)

Media & Playback

Apps for playing audio, video, and handling media codecs.

VLC Media Player

VLC is a free, open-source media player that supports virtually every audio and video format without needing additional codecs. It can play files from local storage, discs, network streams, and webcams.

Key Features
  • Plays almost any media format (MKV, MP4, AVI, FLAC, MP3, and more)
  • DVD and Blu-ray playback support
  • Network stream playback (HTTP, RTSP, MMS)
  • Subtitle support with customizable appearance
  • Basic audio and video conversion
  • No spyware, ads, or user tracking

K-Lite Codec Pack Mega

K-Lite Codec Pack Mega is a comprehensive collection of codecs, filters, and tools that enables Windows and its built-in apps to play virtually any audio or video format. It also includes Media Player Classic – Home Cinema (MPC-HC) as a standalone player.

Key Features
  • Adds support for H.264, H.265, AV1, VP9, and hundreds of other formats to Windows
  • Includes MPC-HC — a lightweight, feature-rich media player
  • Hardware-accelerated decoding (DXVA2 / D3D11) for smooth 4K playback
  • Subtitle rendering with MadVR support
  • Seek preview (thumbnail scrubbing) in MPC-HC
  • Unattended/silent installation — pre-configured out of the box

HEVC Video Extensions

The HEVC Video Extensions package adds native H.265/HEVC decoding support to Windows 10. It enables smooth playback of high-efficiency video files — including 4K and HDR content — directly in the Films & TV app and other supported players, without requiring third-party codecs.

Key Features
  • Native H.265/HEVC video decode support in Windows 10
  • Hardware-accelerated decoding on supported GPUs for reduced CPU load
  • 4K Ultra HD and HDR10 video playback
  • Seamless integration with the Windows Films & TV app
  • Reduces file sizes by up to 50% compared to H.264 at the same quality
  • Required by many modern cameras, drones, and mobile devices that record in HEVC

Files & Utilities

Apps for managing, searching, compressing, previewing, and converting files.

7-Zip

7-Zip is a free, open-source file archiver with a high compression ratio. It integrates into Windows Explorer via a right-click context menu, making it fast and easy to compress or extract files without opening a separate application.

Key Features
  • Creates and extracts 7z, ZIP, TAR, GZIP, BZIP2, XZ, and more
  • Extracts RAR, ISO, CAB, MSI, WIM, and many other formats
  • AES-256 encryption for both 7z and ZIP archives
  • High compression ratio — often better than ZIP or RAR
  • Windows Explorer shell integration (right-click menu)
  • Completely free with no ads or restrictions

WinRAR

WinRAR is a powerful archive manager that supports the RAR and ZIP formats natively, as well as unpacking many other archive types. It is the go-to tool for creating RAR archives and repairing damaged ones.

Key Features
  • Create and extract RAR (including RAR5) and ZIP archives
  • Extract over 15 archive formats (7z, TAR, GZ, ISO, CAB, and more)
  • Repair damaged archives with recovery record support
  • Create self-extracting (SFX) archives — no WinRAR needed to extract
  • Password protection and AES-256 encryption
  • Split archives into multiple volumes for easier sharing

Everything

Everything is an ultra-fast file search utility for Windows that indexes the entire NTFS file system in seconds. Unlike Windows Search, results appear instantly as you type — no waiting, no indexing delays.

Key Features
  • Indexes the full file system almost instantly on first launch
  • Real-time updates — new files appear in results immediately
  • Supports wildcards, regex, and advanced search filters
  • Filter by size, date, path, extension, and more
  • Accessible via keyboard shortcut from anywhere
  • Extremely lightweight — uses minimal CPU and memory

File Converter

File Converter is a simple but powerful tool that adds a "File Converter" option to your right-click context menu. Select one or more files, choose a target format, and the conversion will begin.

Key Features
  • Convert audio: MP3, AAC, FLAC, OGG, WAV, and more
  • Convert video: MP4, MKV, AVI, WebM, GIF, and more
  • Convert images: JPG, PNG, WebP, ICO, and more
  • Batch convert multiple files at once
  • Powered by FFmpeg — reliable and high quality output
  • Fully integrated into Windows Explorer right-click menu

QuickLook

QuickLook brings macOS-style file previews to Windows. Select any file in Explorer and press Space to instantly preview its contents — no need to open the file in a separate application.

Key Features
  • Preview images, videos, audio files, and animated GIFs
  • Preview PDFs, Word documents, Excel spreadsheets, and PowerPoint files
  • Preview ZIP archives, source code files, and Markdown
  • Extensible via plugins (pre-installed on Windows 10 25H2 Hillcom Edition)
  • Press Space to open, Space again or Esc to close
  • Runs silently in the system tray — always available
