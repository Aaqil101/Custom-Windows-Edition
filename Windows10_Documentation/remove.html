Remove Documentation

This documentation is stored locally on your PC at C:\Windows10_Documentation. It is safe to delete at any time — removing it will not affect any installed software.

Option 1 — File Explorer
  1. Open File Explorer and navigate to C:\
  2. Right-click the Windows10_Documentation folder
  3. Click Delete and confirm
Option 2 — Command Prompt

Open Command Prompt and run:

rmdir /s /q "C:\Windows10_Documentation"
Option 3 — PowerShell

Open PowerShell and run:

Remove-Item -Recurse -Force "C:\Windows10_Documentation"
Note: Deletion is permanent. Make sure you no longer need this documentation before proceeding.
